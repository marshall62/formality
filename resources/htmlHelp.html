HTML and Hint Tag Help

HTML Help

Use this tag for images: 

<IMG SRC="/4mality/media/images/14bandGraph.jpg" BORDER=0>

The red part is the file name:  14bandGraph.jpg 
Notice that the question id, 14, is placed at the beginning of 
the filename. This makes the file name unique- a requirement.
(You can get the id from the question editor). 
Just copy the tag above and paste it into the question
stem in the editor. Then substitute only the name of your file. 
Send me the file, I'll put it in it's proper place and voila, 
it will be a masterpiece. Note: you can add HEIGHT and WIDTH
attributes to the image tag with pixel values to adjust the
size of the image, however, the best practice is to create
the image to be the size you expect it to be on the page.

Popup windows (like the one you are reading) are easy, but see me.
Use this "tag" to call a popup window:

<a HREF="javascript: extraWindow('/4mality/media/html/23hint4d.html')">Your text</a>

Just put in the correct html file name and 
the text you want to appear on the page.
Send me the text, images, etc. you want in the popup.
You can also use the template file: extraWinTemplate.txt

This tag creates a line break:  <br>
You can use several of them to get more vertical space: 
<br><br><br>

These six characters make the browser add a horizontal space: 
&nbsp;
You can use several of them to get more horizontal space: 
&nbsp;&nbsp;&nbsp;&nbsp;

These are font style tags:  bold=<b>  italic=<i> 
Note: you must supply closing tags (using the slash /) so the 
browser knows when to stop applying the style. For example:   
...it was the way he said it...
looks like this in HTML:  
..it was the <i>way</i> he <b>said</b> it...

The underline tag is: <u>undelined text</u>
undelined text

Hint Tags

Each hint has a specific tag that identifies the text that should be 
highlighted with the appropriate color when a specific hint is 
selected. The hint tags work just the HTML font style tags: there is 
a beginning tag and a closing tag. Whatever is between the tags will 
be highlighted. For example, the words "Gordon Stadium" and "79,407" 
in this marked up text:

{h3a}Gordon Stadium{/h3a} can seat {h3a}79,407{/h3a} people,

will appear highlighted when the hint at level 3 for coach a is 
selected.
Note: coaches are lettered a, b, c, d. 

An easy way to apply tags is to go to the hint edit page. You get 
there by clicking on the hint ID in the hint table on the question 
editor page. That hint's tag will appear on the form. Copy it, go 
back to the question edit page and paste the tag in the appropriate 
place(s).

You can put any number of hint tags around the text. The inactive 
tags will be ignored. For example, the text "79,407" in these lines:

{h3a}Gordon Stadium{/h3a} can 
seat {h3a}{h5c}{h1a}79,407{/h1a}{/h5c}{/h3a} people,

is marked up to be highlighted for three different hints. 
Note: the order of hint tags does not matter, but you must 
provide a closing tag for each opening tag- or bad things will happen!!!
Test your question's appearance by selecting the "View Question" link
on the question edit page.
